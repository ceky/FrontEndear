# My tiny blog 
> Implemented using Jekyll

## Build locally
``` bash
jekyll serve --watch --baseurl ""
```

http://localhost:4000/index.html

# deploy to surge.sh

Write ```surge``` and add the project path to /_site_

http://ceky.surge.sh
