# Blog
A blog implemented in Jekyll.

## Build locally
``` bash
jekyll serve --watch --baseurl ""
```

Then open this url: http://localhost:4000/index.html

## Deploy the blog to surge.sh

Write ```surge``` and add the project path to /_site_

http://ceky.surge.sh
